Gestion des risques
Failles identifiées & solutions
Les identifier et les corriger en amont est la marque d'une entreprise sérieuse. Voici les 6 risques majeurs et comment AfriDeal les adresse.
Fraude à la propriété

Un vendeur tente de vendre un produit volé ou emprunté. L'agent se laisse tromper par un faux document.

Solution : CNI + justificatif obligatoires. Documents photographiés et archivés. Doute = refus systématique. Agents formés à détecter les faux.
Agent malhonnête

Un agent collecte un produit mais ne le rapporte pas, ou partage les coordonnées vendeur/acheteur.

Solution : Collecte tracée (photos horodatées, GPS). Infos masquées dans l'app agent. Contrat avec clause de responsabilité. Audit aléatoire.
Vendeur qui contourne

Après soumission, le vendeur vend directement à un acheteur trouvé ailleurs, contournant la commission.

Solution : Après collecte physique, le produit est chez nous. Accord formel signé avant collecte. Aucune marge pour contournement.
Désaccord état produit

L'acheteur affirme que le produit est en moins bon état que décrit. Risque de dispute et mauvaise réputation.

Solution : Photos officielles horodatées font foi. Réclamation 24h post-livraison. Remboursement partiel/total si preuve photo justifiée.
Scalabilité agents

Demandes augmentent trop vite — agents insuffisants, délais s'allongent, expérience dégradée.

Solution : Priorité missions par valeur/délai. Agents freelance certifiables. Démarrage Douala uniquement puis expansion progressive.
Double paiement Mobile Money

Réseau instable → double tap → risque de double débit ou double commande enregistrée.

Solution : ID unique de transaction généré côté serveur avant paiement. Doublon détecté = une seule transaction. Jamais de double débit.
